# 🌐 Language Translator Web App

A clean and simple web application that allows users to translate text between different languages quickly and efficiently.

---

## 📌 Overview
This project provides an easy-to-use interface for translating text, focusing on simplicity, responsiveness, and usability.

---

## ✨ Features
- 🌍 Multi-language text translation
- ⚡ Fast and responsive interface
- 🖥️ User-friendly design
- 📱 Mobile and desktop compatible

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

---

## 📂 Project Structure
Language-transulator/
│── index.html
│── main.css
│── script.js
│── assets/

---

## ⚙️ Installation & Setup
1. Clone the repository  
   git clone https://github.com/Sowmya-116/Language-transulator.git  
2. Open the project folder  
3. Open index.html in your browser  

---

## 🚀 Usage
1. Enter text  
2. Select source language  
3. Select target language  
4. Click "Translate"  
5. View output
